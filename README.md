# Unraid User Scripts

## Scripts

- [**Shutdown All Docker Containers**](./scripts/shutdown-containers/README.md)

## Usage

1. Install the `User Scripts` plugin from the **Unraid Community Applications**.
2. Click the `Add Script` button and **enter a name for your script**.
3. Click on `Edit Script` from the settings wheel and **paste the script into the text area**.
4. Click on `Save Changes`.
